# Imágenes Decorativas vs. Informativas

Para determinar si una imagen debe tener un texto alternativo descriptivo o un `alt=""`, evalúa su propósito.

## Imágenes Decorativas (`alt=""`)
Una imagen es puramente decorativa si se cumple **alguna** de las siguientes condiciones:
1. **Redundancia Visual**: La imagen es una ilustración de texto adyacente y el texto por sí solo transmite todo el significado (ej. una foto genérica de un avión al lado del texto "Reserva tu vuelo").
2. **Propósito Puramente Estético**: La imagen existe solo para mejorar la presentación visual (bordes, esquinas decorativas, separadores, texturas).
3. **Ambiente Visceral**: Fotografías de "stock" o de relleno que no contienen información que el usuario deba comprender para completar una tarea o entender el contenido principal.
4. **Parte del Diseño**: Íconos que acompañan a un texto pero no son el único mecanismo para entender la acción (ej. Un ícono de "Casa" al lado de la palabra "Inicio" en un menú).

## Imágenes Informativas / Funcionales (`alt="..."`)
Requieren descripción obligatoria:
- Aportan datos críticos que no están en el texto circundante.
- Muestran procesos o instrucciones importantes.
- Son el único contenido de un enlace o botón funcional.
